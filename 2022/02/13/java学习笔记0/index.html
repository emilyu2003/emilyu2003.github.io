  • UML? VISIO? UMLet工具和Dia？

二 java的表达式，流控制和数组

和C的区别
  • 强类型语言
  • 不提供指针运算符！但是有引用
  • 字符串运算符+
  • 零填充的右移>>>
  • ++、–是最快的操作，还有移位操作，都是一个时钟周期的事情
混合类型运算
  • long：都不是float和double，至少有一个是long
  • int：只能是int
  • double：至少有一个是double
  • float 只少有一个是float，且都不是double
操作符
  • 位移
    • 无符号右移：>>>
  • 逻辑运算（区别于布尔运算）
    • &|^~
    • 异或曾经被大量用在画画中
  • 逻辑操作符
    • && || ! & | ^
  • 赋值
    • 注意新增一个>>>=
  • 其他
    • instanceof：
      • op1 instanceof op2
类型强制转换

整型与浮点型

流控制
数组声明
  • 可以改变空间 用new
  • 在java里面数组得名字是一个地址变量（区别于c是地址常量）

三 java语言的对象与类

封装
多态
继承
创建对象的步骤
  • 声明
  • 实例化
  • 初始化
对象的回收
  • 当把一个对象没有引用指向它时，会被自动回收
类的构造方式
类成员方法定义
数据的隐藏和分装
  • 数据隐藏：private，保证对象中数据的一致性
  • 封装
    • 隐藏类中具体实现的细节
    • 强迫使用统一接口访问数据
    • 使代码可维护性好
this 引用
  • 指向当前的对象本身
重载
  • 在同一个类中一个方法名被用来定义多个方法
  • 参数必须不同
  • 返回类型、修饰符可相同
激活重载的构造方法
子类
  • 有父类全部的属性和自己新定义的属性
  • 带private修饰符的不可被继承
  • 构造方法不能被继承但可以调用，调用自己的用this()，调用父类的用super()
  • extends
  • java中任何一个子类的实例化都可以作为父类的实例使用，可以调用父类具有的方法 -> 所以一个类数组的每一个单元的类型可以是各种子类
  • 单继承，即extends后面只有一个类名
多态
  • 引用类型 + 实例化类型
  • 父类对象的变量作为子类对象的变量使用
重写
  • 子类可以改变从父类继承的行为，被重写方法的返回值、方法名、参数列表要与父类中的方法完全一样
  • 如果不一样，那叫多态
  • 方法的访问权限不能缩小
抽象类
  • 抽象类不能直接用来生成实力，一般可通过定义子类进行实例化
接口
  • interface
  • 所有成员方法都是抽象的，并且只能定义static final成员变量
  • 接口的继承（？ 用implements，该子类中必须实现接口（及其超类）中的所有方法
  • extends和implements同时存在时，就按这个顺序写：extends <> implements <>
  • 可以通过实现接口实现多重继承：一个类可只继承一个父类，并实现多个接口
静态变量类型
  • static / class variable
静态类方法
  • 可以直接被调用，而不需要生成任何实例
final
  • 被定义成final的类不能被继承（不能有子类）
  • 成员方法不能被重写，成员变量不能被改变，一般用大写表示，是常量

四 java包与访问控制、例外处理

包与访问处理可以和using namespace std类比

类分组
  • 语句：package 包名
高级访问控制
  • 不针对类，以类为单位
  • 应用于class或type层次
内部类
  • wrapper：将基本类型表示成类，可调用MIN_VALUE和MAX_VALUE，还有valueOf和toString实现字符串与数字之间的转化
collection API 集合类
  • set
    • 不能包含重复元素
    • HashSet
    • TreeSet
  • list
    • 有序集合，元素可重复
  • string
捕捉与例外处理
  • try，后面必须跟随一个catch或者finally语句块
  • catch
  • finally
  • 抛出：throw
  • 获得例外并处理：connectMe

五 I/O

  • 设备、DMA
流式I/O
  • 按输入源头
    • node stream
    • filter stream
  • 字节流
    • input stream
      • 甚至可以skip
    • output stream
      • void write * 3
      • void flush() //强行写，保证写进去了
  • reader: int read()
  • writer: int write()
  • grep
  • 管道流：把一个线程的输出连接到另一个线程的输入
    • 倒排字典 押韵 -> 好背 lol
    • 单词逆序 -> 排序 -> 然后再逆回来
  • 过滤流：从原始流成块读入，或将数据累计到一个大数据块儿后再成批输出
大作业 orz

六 AWT及Applet编程

  • AWT 抽象窗口工具集
  • Java.awt包
容器
  • 所有组件都必须放在容器内（如窗口）
  • frame / panel / layout manager（如比较古早的网页布局设计方法）
  • events 描述不同类型的用户动作
  • component
    • 支持基本的drawing
    • paint(), update(), repaint()
applet
  • html 标志方法
